## scanf
1) 공백이나 엔터도 아스키코드값으로 다 받아드림
2) 변환문자를 공백으로 구분하면 공백제외하고 입력받음

###
빠른 순서

#버퍼 : 임시저장공간 (캐쉬메모리,레지스터)

메모리

저장장치 (SSD, HDD)

#버퍼

주변장치 (키보드,마우스, 스캐너)

변수 (char int)에 아무것도 없으면 \n 입력받을때까지 입력값을 버퍼에 저장
읽어올때 자료형을 보고 버퍼에서 하나씩 가져옴

stdin : 키보드로 입력받은 버퍼
stdout : 표준출력

##  언어 문법관련

1. 지역변수

-함수(블럭) 내에서 선언된 변수
-함수가 호출되고 종료되면 메모리에서 자동 해제
-디버깅에 유용하다

2. 전역변수

- 함수 밖에서 선언된 변수
- 프로그램 종료시까지 유지되는 변수
- 이름이 바뀌거나 값이 이상하면 모든 함수를 살펴야 한다
- 같은 이름의 지역 변수에 의해 사용 범위가 제한됨

3. 정적 지역변수(static)

- 함수 내에서 선언되지만 유효범위는 프로그램 전체(전역)

4. 레지스터 변수

- CPU내의 레지스터에 할당 되는 변수
- 메모리상의 주소가 없음
- 사용 여부는 컴파일러가 결정함

gcc -c 파일제목
gcc -o 실행파일제목 파일제목
